    HIGHLIGHTS OF CLOUDWAVE’S MULESOFT MEETUP AT WASHINGTON DC

    CloudWave a reliable MuleSoft Partner and System Integrator hosted its 3rd MuleSoft Meetup successfully in Washington DC on 24th October and this meet was wrapped up some interesting topics with demos where key attendees were from Industries like Government, Finance, and Retail.

    Host for the event was Ravindra Bhati – Mulesoft (Solutions Engineer) and Prabhakar Ramakrishnan, President – CloudWave. This meet intended for connecting Developers together to discuss new features and tools developed by MuleSoft Partners.

    Agenda:

    • 6:00 PM: Registration
    • 6:30 PM: Mule Watch – Performance Monitoring Tool for Developers
    • 7:00 PM: Run Time Fabric
    • 7:30 PM: Wrap-up / Networking

    As stated in the agenda CloudWave’s innovative solution MuleWatch Connector and Runtime Fabric Overview by MuleSoft Architects were depicted. Prabhakar, demonstrated MuleWatch Connector and case study about the uses of Mule Watch and that got greater response from attendees. The Mule Watch Connector runs On-Prem or On Cloud-Hub, helps to monitor any Mule application in real time and using which developers can resolve performance problems faster than ever before and check transactions in real time to fix anomalies pro-actively

    After that Ravindra, explained how deployment and orchestration of Mule applications and API gateways can be automated by a container service called Anypoint Runtime Fabric. It runs within a customer-managed infrastructure on AWS, Azure, virtual machines (VMs), and bare-metal servers. Some of the key capabilities of Anypoint Runtime Fabric like Scaling applications across multiple replicas, Running multiple versions of Mule runtime on the same set of resources, Automated application fail-over and Connecting Runtime Fabric to Anypoint Management Center were discussed.

    The event was very much interactive and attendees discussed on topics for next meetup. On top of everything, 3 lucky trivia questions – winners walked away with a free MuleSoft training and certification vouchers
